# RAD: Rad Audio Download Manager

Audio file download manager supporting various audio hosting sites.

Built with [Tauri](https://tauri.app/) and [Svelte](https://svelte.dev).

Currently supported hosts:

- Soundgasm
- Whyp
- Vocaroo
- Audiochan
- Erocast
