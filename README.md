# RAD: Reddit Audio Download Manager

Audio file download manager from various audio hosting sites commonly used
on many subreddits.

Built with [Tauri](https://tauri.app/) and [Svelte](https://svelte.dev).

Currently supported hosts:

- Soundgasm
- Whyp
- Vocaroo
